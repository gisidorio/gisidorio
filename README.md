## 📌 Projetos em destaque

### 🛒 CandyHill - E-commerce

Sistema completo de e-commerce com arquitetura limpa, testes automatizados e integração com Inteligência Artificial.

---

### 🔹 Backend (.NET API)

API REST desenvolvida com foco em boas práticas e escalabilidade:

- Arquitetura em camadas (Clean Architecture)
- Aplicação de SOLID e Clean Code
- Autenticação com JWT
- CRUD de produtos, pedidos e usuários
- Integração com SQL Server (Stored Procedures + TVP)
- Testes Unitários (regras de negócio)
- Testes de Integração (persistência e banco de dados)
- Integração com Agentes de IA (recomendações)

🔗 https://github.com/gisidorio/ecommerce-candy-hill-web-API

---

### 🎨 Frontend (Angular)

Aplicação web para consumo da API:

- Desenvolvido com Angular
- Integração com API REST
- Listagem de produtos
- Carrinho de compras
- Consumo de recomendações com IA
- Organização de componentes e serviços

### 🧠 Diferenciais do Projeto

- Uso de Inteligência Artificial no contexto de e-commerce
- Separação clara entre backend e frontend
- Estratégia de testes baseada em boas práticas de mercado
- Código organizado, escalável e preparado para evolução
